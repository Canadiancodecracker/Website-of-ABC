# 2026 Spring Festival News Page - Image Setup

## Image Required

The news pages reference an image at:
```
assets/images/2026-spring-festival-food.jpg
```

### Steps to Add the Image:

1. Save the festive food photo (the overhead view of the meal containers) as a JPG file
2. Place it in: `/Users/mingshen/Website-of-ABC/assets/images/`
3. Name it: `2026-spring-festival-food.jpg`

### Alternative:
If you want to use a different image name or location, update both news pages:
- `news/2026-02-17-spring-festival-celebration.html` (line ~22)
- `news/2026-02-17-spring-festival-celebration-zh.html` (line ~22)

## Quick Command to Copy Image:

If your image is on Desktop:
```bash
cp ~/Desktop/[your-image-name].jpg assets/images/2026-spring-festival-food.jpg
```

If it's in Downloads:
```bash
cp ~/Downloads/[your-image-name].jpg assets/images/2026-spring-festival-food.jpg
```

## Verification:

After adding the image, open either news page in a browser to verify it displays correctly.
