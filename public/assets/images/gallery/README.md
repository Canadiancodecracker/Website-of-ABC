# Factory Gallery Images

Please place your factory gallery images in this directory with the following filenames:

1. **raw-material-area.jpg** - Large raw material silos, underground unloading points
2. **reaction-section.jpg** - Reaction tanks, operator platforms, temperature control systems
3. **drying-packaging-system.jpg** - Fluidized bed dryers, fully automated packaging lines
4. **qc-qa-lab.jpg** - Testing equipment, spectrometers, lab technicians at work
5. **warehouse-logistics.jpg** - Finished goods pallet zones, forklift operations, truck loading and dispatch
6. **ehs-safety-facilities.jpg** - Sprinkler stations, exhaust gas treatment systems, staff safety training scenes

## Image Requirements:
- Recommended size: 800x560 pixels (or 4:3 aspect ratio)
- Format: JPG, PNG, or WebP
- File size: Optimized for web (under 500KB per image recommended)

The images will be automatically displayed in the Factory Gallery page once placed in this directory.
